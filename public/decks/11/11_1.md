# なぜ"見抜く力"が要るのか

---

# 3年で、ここまで来た

AI生成「**ウィル・スミスがスパゲッティを食べる動画**」

- **2023年**：顔が溶ける・崩れる → 一目で"ニセモノ"
- **2026年**：ほぼ実写と見分けがつかない

→ この3年でAIは激変。静止画も同じ ― **昔の派手な破綻は減った**
→ だからこそ **残った"微妙なホコロビ"を見抜く目** が要る

[https://x.com/i/status/2013893267545563624](https://x.com/i/status/2013893267545563624)
参考：[Will Smith Eating Spaghetti test（Wikipedia）](https://en.wikipedia.org/wiki/Will_Smith_Eating_Spaghetti_test)

---

# 昔のAIと、今のAI

| | 破綻 |
|---|---|
| **昔のAI** | 指が6本／文字がぐにゃぐにゃ ＝ 一目でバレた |
| **今のAI（GPT Images 2.0）** | 手も文字もかなり上手い ＝ 派手な失敗は激減 |

→ 残るのは **"パッと見では気づかない微妙な破綻"**

# だから"見抜く目"が価値を持つ

- 誰でも **"それっぽい1枚"** は作れる時代
- 差がつくのは、**残った微妙な破綻を見抜いて直せるか**
- これは **納品前の品質管理（QC）** ― プロの仕事

> AIが上手くなるほど、**"見抜く目"の価値は上がる**

# AIはなぜ破綻するのか

---

# AIは"世界のルール"を持たない

- AIは **学習したパターンから"局所的にそれっぽい"** 画像を作る
- でも人間のような **"世界のルール"を持っていない**：
  - 「手は指5本」という **解剖** を知らない
  - 文字を **"意味"ではなく"線と形"** として描く
  - 「建物は構造的な正しさの上で成立する」という **物理** を持たない

→ 質感は得意。でも **数・文字・構造・全体のつじつま** を外す
→ その穴を埋める **"世界の知識"を持つのは人間**

参考：[なぜAIは文字や数が苦手か（The Conversation）](https://theconversation.com/if-ai-image-generators-are-so-smart-why-do-they-struggle-to-write-and-count-208485)

# 破綻カタログ
## 5つのカテゴリで見る

---

# 破綻の5カテゴリ

| カテゴリ | 破綻の例 |
|---|---|
| **① 数と体** | 指の融合／左右で違う目・ピアス |
| **② 文字・記号** | 意味のない文字列／崩れた看板 |
| **③ 質感・光** | 肌がツルツル／影の向きが矛盾／切り抜き貼り付け風 |
| **④ 構造・物理** | 物が融合／支えのない構造／鏡・水面が合わない／パース破綻 |
| **⑤ 繰り返し・整合** | 窓・群衆がコピペ反復／時代がちぐはぐ |

参考：[AI画像を見抜く5つの兆候（Northwestern Kellogg）](https://insight.kellogg.northwestern.edu/article/ai-photos-identification)

---

# どこに出やすいか

- **背景（第10回素材）** → **④ 構造・物理 ／ ⑤ 繰り返し・整合**
  - 建物の歪み・窓の反復・ありえない構造・パース
- **キャラ（第9回素材）** → **① 数と体 ／ ③ 質感・光**
  - 手元・装飾の途切れ・髪・光のちぐはぐ

→ 自分の素材の **種類に合わせて重点を変える**

---

# 見抜く方法：引いて見る／寄って見る

- **引いて見る**（縮小・目を細める）
  - 全体の違和感を一瞬で感じ取る（構図・パース・光）
- **寄って見る**（100%ズーム・部分拡大）
  - 細部の破綻を確認（指・文字・繰り返し・境界）

→ 順番は **全体 → 構造 → 細部**
→ **印刷して見る** のも有効（画面で見逃す粗が、紙にすると分かる）

---

# 自分の絵は自分で気づけない

- 自分の絵の破綻は **脳が勝手に補完** してしまう
- **他人の目** を通すと一瞬で見つかる
- ペアで見合う ＝ **校正・レビューの原理**

→ ワークでは **ペアで相互レビュー** する
