# 直す道具②

# 手塗りの基礎

新規レイヤーに描く（元画像は触らない）

- **スポイト** で周囲の色を拾う → その色で塗る
- **ブラシ／ペン** で形を整える／描き足す
- **ぼかし・不透明度** でなじませる

→ **ツールは慣れた方でOK**：**クリスタ** or **Photoshop**
→ マウスは「選択 → 塗りつぶし → ぼかし」で代替

---

# クリスタと組み合わせる

**描く直し**（キャラの手・顔など）は、慣れた **クリスタ** が強い

1. Photoshopで **書き出し**
2. **クリスタで手塗り**（ブラシ・ペン）
3. Photoshopに戻して **調整レイヤーでなじませ**

→ 「非破壊のハブ＝Photoshop、描くのは得意なツール」で使い分け
→ 往復は時間がかかる。**授業内はPhotoshop完結でもOK**（クリスタ仕上げは放課後でも可）

---

# AIで埋める所／手で直す所

| | 得意 | 苦手 |
|---|---|---|
| **AIで埋める** | もっともらしく埋める・速い・背景補完 | キャラの一貫性（顔・服）／狙った形 |
| **手塗り** | 一貫性・狙った形・細部 | 広い面を速く埋める |
| **調整レイヤー** | 色・光を非破壊でなじませる | 形そのものは直せない |

→ **埋めるはAI／形は手／色・光は調整レイヤー**

---

# なじませが命

直した所が **浮かない** ように、周囲に合わせる

- **色・明度・トーン**：**調整レイヤー**（クリッピング）で非破壊に合わせる
- **エッジ**：境界をぼかす（くっきりしすぎない）
- **粒状感・質感**：周囲に合わせる／レイヤースタイルで影・光

→ 例：**キャラと背景で夕日の当たり方を合わせる**（第10回の整合）
→ **調整レイヤーなら非破壊で何度でも** 直せる。**なじませて初めて「直した」と言える**
