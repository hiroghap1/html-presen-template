# プロは設定資料で一貫性を担保する

### キャラクター設定資料（ターンアラウンド）
- 正面・横・斜め・**後ろ** ＝ 三面図／四面図
- アニメ・ゲームの **作画の基準** に使われる

### プロが守っていること
- **シルエット**（黒く塗っても区別できる）
- **髪型・装飾** の特徴
- **色のパターン**（メインカラー3色程度）

→ 「1枚絵の完成度」より **素材としての一貫性** を優先

---

# 今日作るもの

| 区分 | 内容 | 必須/任意 |
|---|---|---|
| ターンアラウンド | 正面・横・斜め45度・**後ろ姿** | 必須（4面） |
| アクション | 走る／振り向く／座る／構える 等 | 必須（1枚） |
| 表情差分 | 笑う／驚く／怒る 等 | 任意 |

→ **後ろ姿はGPT Image 2なら作れる**ので、今日は **4面** を狙う

---

# アクションポーズのボキャブラリー

| 区分 | 言葉 |
|---|---|
| 立ち姿 | 立つ・腕組み・片足重心 |
| 動き | 走る・歩く・振り向く・ジャンプ |
| 感情表現 | 笑う・怒る・驚く・考える |
| 戦闘・スポーツ | 構える・武器を持つ・蹴る |

→ キャラの **性格に合うもの** を1つ選ぶ

---

# 一貫性は3点で寄せる

### 参照画像 × プロンプト × 反復

- **参照画像**：第8回のベスト1枚を添付し「この顔・この服で」と伝える
- **プロンプト**：服・髪型・体型を毎回 **同じ言葉** で指定する
- **反復**：1発で決めず、揃うまで2〜3回出す

> 完全一致は無理。**作品として成立するレベル** を狙う

# ワーク②
## GPT Image 2でポーズを量産する

---

# 量産プロンプト①：ターンアラウンド

```
# 被写体
自分のキャラ（第8回のベスト1枚を参照画像に添付）

# 構成
同一キャラのターンアラウンドシート。
正面 / 横 / 斜め45度 / 後ろ姿 を横一列に等間隔で並べる

# 条件
全ポーズ同じ服・髪型・体型・身長・表情
背景は無地／全身が収まる／キャラは1人だけ
```

→ まとめて1枚で出す。崩れたら反復して整える

---

# 量産プロンプト②：アクション

```
# 被写体
同じキャラ（同じ参照画像を添付）

# ポーズ
走っている全身（横から）  ← ここだけ差し替える
例：振り向く／椅子に座る／武器を構える

# 条件
服・髪型・体型はターンアラウンドと同一
背景は無地／全身が収まる
```

---

# 一貫性チェックの見方

黒く塗りつぶしても **同じキャラに見えるか**

| 見る点 | OKの目安 |
|---|---|
| シルエット | 髪型・体型の輪郭が4面で揃う |
| 髪型・装飾 | 特徴的なパーツが消えていない |
| 服 | 色・形が同じ |
| 色味 | メインカラーがブレていない |

→ 全部完璧でなくてOK。**主役として成立すれば合格**
