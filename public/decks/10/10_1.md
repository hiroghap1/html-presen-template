# 背景は別の専門職が作る（通常のフロー）

### 美術設定／背景美術／コンセプトアート
- **キャラ作画とは別の専門職**（背景美術スタッフ）が担当
- **キャラとは別レイヤー** で、世界の質感・光・時代を設計

### 必要な情報
- **場所と時代**（いつ・どこの世界か）
- **光と色のトーン**（朝／夕／夜・暖色／寒色）
- **象徴物**（建築・小道具・植生）

→ AIでも同じ：**背景はキャラと分けて、素材として作る**

# 世界観を言語化する5つの問い

| 問い | 観点 | 例 |
|---|---|---|
| **どこ（場所）** | 都市／自然／室内／ファンタジー | 海沿いの旧市街 |
| **いつ（時代・季節）** | 現代／近未来／中世風／和風 | 近未来・夏の終わり |
| **光（時間帯・天候）** | 朝／昼／夕／夜・晴／曇／雨 | 夕方・薄曇り |
| **雰囲気（トーン）** | 賑やか／静か／不穏／温かい | 静かで少し寂しい |
| **象徴物** | 建築・小道具・植生・乗り物 | 錆びた看板・路面電車 |

→ 5つが埋まると、**キャラとズレない背景プロンプト** が書ける

参考：[ゲームの「世界観」とは？（Confidence Creator）](https://confidence-creator.jp/column/25866/)

# 背景の構図をコントロールする

第7回の構図コントロールを **背景に応用する**

- **画角・縦横比**：横長（16:9 / landscape）で空間の広がりを出す
- **引き・広角**：寄りすぎない。空間全体を見せる
- **抜け／余白**：後で **キャラを置くスペース** を空けておく
- **キャラは入れない**（人物なしの空間として出す）
  - 人物を入れると第9回のキャラとほぼ別人になる → 一貫性が崩れる

→ 背景は背景、キャラはキャラで **分ける**
